# DotFiles

I don't want to loose all of my useful confs so I created this repo.

### Basic components:
- Browser: Firefox
- Shell: zsh (Oh My Zsh)
- Terminal: alacritty (with Nerd Fonts)
- Multiplexer: tmux
- Text editor: NeoVim

### Additional:
- Window manager: i3gaps
- File manager: vifm
- System information: conky
- App runner: rofi
- Wallpapers: nitrogen
- Screen settings: arandr
- Audio settings: alsamixer

### Development:
- JS: nvm
- Python: pyenv, pipenv
