# DotFiles

I don't want to loose all of my useful confs so I created this repo.

### Basic components:
- Browser: Firefox
- Shell: zsh (Oh My Zsh)
- Terminal: alacritty (with Nerd Fonts)
- Multiplexer: byobu
- Text editor: NeoVim

### Additional:
- Window manager: i3gaps
- File manager: ranger
- App runner: rofi
- Wallpapers: nitrogen (feh)
- Screen settings: arandr (xrandr)
- Audio settings: alsamixer

### Development:
- JS: nvm
- Python: pyenv, pipenv
- Haskell: stack
