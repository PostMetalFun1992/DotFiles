# DotFiles

I don't want to loose all of my useful confs so I created this repo.

### Basic components:
- OS: Arch Linux
- Shell: zsh (Oh My Zsh)
- Terminal: kitty (FiraCode Nerd Font)
- Terminal multiplexer: tmux (with tpm)
- Window manager: i3-gaps
- Status bar: polybar (Iosevka Nerd Font)
- App launcher: Rofi
- File manager: Ranger
- Browser: Firefox
- Text editor: NeoVim
- AUR Helper: yay
